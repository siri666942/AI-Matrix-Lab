# AI Matrix Lab - 快速启动

## 🚀 立即体验

### 方式1：Mock模式（推荐新手）

1. 打开应用
2. 在"自然语言输入"框输入：`rotate 45`
3. 点击"转换"按钮
4. 观察右侧的变换效果！

### 方式2：预设变换

1. 直接点击左侧的预设按钮
2. 例如：点击"旋转90°"
3. 立即看到效果

### 方式3：手动输入矩阵

1. 在"矩阵输入"区域修改数值
2. 例如：将第一行第二列改为 1
3. 实时看到剪切变换

## 📚 支持的Mock命令

- `rotate 45` - 旋转45度
- `rotate 90` - 旋转90度
- `reflect x` - X轴反射
- `reflect y` - Y轴反射
- `scale 2` - 缩放2倍
- `shear 1.5` - 剪切变换
- `squeeze` - 挤压变换

## 🎯 学习路径

### 初学者
1. 点击所有预设按钮，观察效果
2. 尝试修改矩阵数值
3. 切换不同图形类型

### 进阶用户
1. 使用Mock模式尝试不同命令
2. 观察行列式和特征值的变化
3. 理解特征向量的几何意义

### 高级用户
1. 配置OpenAI API
2. 使用自然语言描述复杂变换
3. 探索矩阵组合效果

## 🔧 配置OpenAI API（可选）

1. 关闭"Mock AI模式"开关
2. 输入API密钥：`sk-...`
3. （可选）修改API地址：
   - OpenAI: `https://api.openai.com/v1`
   - DeepSeek: `https://api.deepseek.com/v1`
   - Moonshot: `https://api.moonshot.cn/v1`

## 💡 提示

- 按Enter键可快速提交自然语言输入
- 行列式为0表示矩阵不可逆
- 绿色箭头只在特征值为实数时显示
- 切换图形类型可以更好地理解变换

## ❓ 常见问题

**Q: 为什么看不到绿色箭头？**
A: 特征值为复数时不显示特征向量。尝试反射或缩放变换。

**Q: Mock模式够用吗？**
A: 对于学习基础概念完全够用！真实AI模式适合探索更复杂的变换。

**Q: 如何重置到初始状态？**
A: 点击"单位矩阵"预设按钮。
